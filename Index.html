どこにでも使える避難用具

自由に移動

火災の情況によって避難場所を変えることができます。またRunawayは様々な窓際で設置することができ、１つのRunawawyで火災の状況によって避難場所を変えられる

安全の場所まで導く

感熱装置

全ての窓に感熱裝置があり、火災時温度が高くない所は感熱裝置が光り、避難者はより早く安全の窓際までRunawawyを使って避難できます。

暗闇の中光る

ライト

火災時は、大抵の空間は暗く、Runawawyはライトがあり、懐中電灯と同じように暗い空間でも導いてくれます。またRunawawyを使って上から降りる際、ライトが光ってる為、消防士がより遭難者を見つけ易く救援し易いです。

How to Use

Install


①窓際或いはパラペットの感熱装置に引っ掛け、設置します。

Lock


②両方の取手を時計回りに回し、締め付けます。

Tear


③ウェビングを下まで剥がします。

Throw


④リールを外側に投下すると、梯子が出てきます。

Wear


⑤着用ベルトを両脇と背中に通し、ベルトを締めます。

Escape


⑥ロープを持って外側に出て、体を壁側に向け、梯子を降ります。
